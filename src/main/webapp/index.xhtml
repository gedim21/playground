JSF 2.2

Developed through the Java Community Process under JSR - 314, JavaServer Faces technology establishes the standard for building server-side user interfaces.

EJB 3.2

Enterprise JavaBeans (EJB) technology is the server-side component architecture for Java Platform, Enterprise Edition (Java EE). EJB technology enables rapid and simplified development of distributed, transactional, secure and portable applications based on Java technology.

JPA 2.1

The Java Persistence API is a Java programming language framework managing relational data in applications using Java Platform, Standard Edition and Java Platform, Enterprise Edition.

Hibernate 4.3

Hibernate an open source Java persistence framework project. Perform powerful object relational mapping and query databases using HQL and SQL.

EhCache

Ehcache is an open source, standards-based cache used to boost performance, offload the database and simplify scalability. Ehcache is robust, proven and full-featured and this has made it the most widely-used Java-based cache.

Elastic Search

ElasticSearch is a distributed RESTful search engine built for the cloud.
